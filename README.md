# nabin-acharya

- **template of:** vite + react + typescript
- **tailwindcss:** v3.x

Website for a Real Estate agent.

This repository contains a small landing page built with Vite, React, TypeScript and Tailwind CSS intended for a real estate agent portfolio and lead capture (hero, contact CTA, and background media).

To run locally:

```bash
npm install
npm run dev
```

Replace the video in `src/videos/` and update content as needed.
